# ChessClone
Creating a desktop application (Chess Clone) using opengl and C++


## Setup

-Start by cloning the repository using : `git clone https://github.com/Mehdibenhadjkhelifa/ChessClone.git`
-Open cmd in the cloned directory.
-run this following command : `vendor\bin\premake\premake.exe TargetIDE` 
with TargetIDE being the IDE to build for 
Example : Replace TargetIDE with vs2022 for to setup for visual studio 2022

Now you can build the solution with no issues and be able to try the application

PS: As of now the application is still in development.
